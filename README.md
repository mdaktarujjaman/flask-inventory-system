# 📦 Flask Inventory Management System

A web-based inventory management system built with Flask.
Supports full CRUD operations, CSV import/export,
and a clean dashboard for managing stock efficiently.

---

## ✨ Features

- 📋 Product CRUD (Create, Read, Update, Delete)
- 📤 CSV Import & Export functionality
- 📊 Inventory dashboard with stock overview
- 🔍 Product search & filter
- 📱 Responsive design with Bootstrap
- ⚡ Lightweight & fast with Flask

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Flask, Python 3.x |
| Frontend | HTML5, CSS3, Bootstrap |
| Database | SQLite |
| Data Export | CSV (Python built-in) |

---

## 🚀 Getting Started

### Installation

```bash
# Clone the repository
git clone https://github.com/mdaktarujjaman/flask-inventory-system.git
cd flask-inventory-system

# Install dependencies
pip install -r requirements.txt

# Start the server
python app.py
```

---

## 📌 Core Features

| Feature | Description |
|---------|-------------|
| Add Product | Add new items with name, quantity & price |
| Update Stock | Edit existing product details |
| Delete Product | Remove items from inventory |
| CSV Import | Bulk upload products via CSV file |
| CSV Export | Download full inventory as CSV |
